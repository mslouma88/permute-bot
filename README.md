# Permute-bot

SERIS Permute Bot est un assistant intelligent pour la gestion des permutations de vacations entre agents de sécurité.

### Fonctionnalités :

• Import automatique du planning PDF

• Détection des vacations jour et nuit

• Gestion des binômes

• Recherche automatique de remplaçants

• Notifications privées Discord

• Validation par chef d'équipe et responsable

• Historique des permutes

### Technologies :

Python
Discord API
SQLite
PDFPlumber

Ce bot simplifie la gestion des plannings et améliore la communication entre agents sur le site INTERXION Marseille MRS2.

Développé pour optimiser les opérations de sécurité et réduire les conflits de planning.
